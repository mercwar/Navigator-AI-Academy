# Prime Agent README

Prime Agent is a modular system designed to evaluate, arbitrate, and synthesize agent behaviors.  
It integrates multiple languages (C, Java, PHP, JSON) to provide a cross-platform framework.

---

## File Structure
- **Evaluation (Java)**  
  - `PrimeMatrix.java` — scoring logic  
  - `PrimeSymbolicCompiler.java` — symbolic compilation  

- **Synthesis (PHP)**  
  - `prime_selector.php` — branch selection  
  - `prime_arbitration.php` — conflict resolution  
  - `prime_synthesis.php` — synthesis pipeline  
  - `prime_patterns.php` — synthesis patterns  
  - `prime_personality.php` — personality traits  

- **Application (C/Java/PHP)**  
  - `prime_agent.c` — runtime agent  
  - `prime_core.c` — core execution  
  - `prime_ai.java` — AI logic  
  - `prime_utils.php` — utilities  
  - `prime_tests.c` — unit tests  

- **Resources + Docs**  
  - JSON: metrics, profiles, governance, rules, dependencies  
  - Markdown: architecture, execution, API, security, tutorial  

---

## Usage
1. Compile Java modules for evaluation logic.  
2. Run PHP synthesis scripts for branch selection and arbitration.  
3. Execute C application files for agent runtime.  
4. Reference JSON resources for metrics, profiles, governance, rules, and dependencies.  
5. Consult Markdown docs for system overview and onboarding.
